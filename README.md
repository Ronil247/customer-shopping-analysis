# Customer Shopping Analysis

## Overview

This project analyzes customer shopping behavior using SQL and Power BI. The analysis focuses on customer demographics, purchasing patterns, product categories, payment methods, shipping preferences, and subscription trends. An interactive Power BI dashboard was created to visualize key metrics and generate business insights.

## Tools Used

* SQL (MySQL)
* Power BI

## Dashboard Preview

![Dashboard](customer_shopping_dashboard.png)

## Analysis Performed

* Customer Demographics Analysis
* Revenue Analysis by Category
* Shipping Type Analysis
* Subscription Status Analysis
* Payment Method Analysis
* Location-Based Analysis
* Purchase Frequency Analysis
* Customer Rating Analysis

## Dashboard Features

* KPI Cards
* Interactive Slicers
* Revenue Breakdown
* Customer Segmentation
* Category Performance Analysis
* Shipping and Subscription Insights

## Key Insights

* Identified the highest revenue-generating product categories.
* Analyzed customer purchasing behavior across different age groups.
* Compared revenue contribution across shipping methods.
* Evaluated subscription trends and customer engagement.
* Examined payment preferences and purchase frequency patterns.
